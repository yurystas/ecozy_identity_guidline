# eCozy Illustrations

This directory contains illustrations, diagrams, and graphic elements used in eCozy documentation.

## Illustration Types

### Brand Illustrations
- Hero images
- Background graphics
- Decorative elements

### Technical Diagrams
- System architecture diagrams
- Network topology diagrams
- Data flow illustrations
- Component interaction diagrams

### Infographics
- Feature comparisons
- Process flows
- Statistics and data visualization

### UI/UX Illustrations
- User journey maps
- Wireframes
- Interface mockups

## File Format

- **SVG** (preferred) - scalable vector graphics
- **PNG** (with transparency) - for raster illustrations

## Naming Convention

`illustration-[type]-[name].svg`

Examples:
- `illustration-system-architecture.svg`
- `illustration-smart-home-ecosystem.svg`
- `illustration-data-flow-diagram.svg`
- `illustration-user-journey.svg`

## Color Guidelines

Illustrations should use colors from the eCozy color palette:
- Primary colors for main elements
- Secondary colors for supporting elements
- Maintain brand consistency

## Usage

```markdown
![System Architecture](/_assets/illustrations/illustration-system-architecture.svg)
*eCozy smart climate control system architecture*
```
