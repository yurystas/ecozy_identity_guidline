# eCozy Icons

This directory contains icon files used across eCozy products and documentation.

## Icon Categories

### Product Icons
- Thermostat icons
- Sensor icons
- Central unit icons
- Mobile app icons

### UI Icons
- Navigation icons
- Action icons
- Status indicators
- System icons

### Social Media Icons
- Social platform icons with eCozy branding

## File Format

All icons should be:
- **SVG format** (preferred for scalability)
- **PNG format** (24px, 32px, 48px sizes for specific use cases)

## Naming Convention

`icon-[name]-[size].svg`

Examples:
- `icon-thermostat-24px.svg`
- `icon-settings-32px.svg`
- `icon-temperature-48px.png`

## Usage

```markdown
![Temperature Icon](/_assets/icons/icon-temperature-24px.svg)
```
