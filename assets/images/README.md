# eCozy General Images

This directory contains general images used throughout the documentation that don't fit into other specific categories.

## Image Types

- Screenshots (UI, app interfaces)
- General diagrams
- Presentation slides
- Marketing materials
- Miscellaneous graphics

## Naming Convention

`[section]-[description].png` or `[section]-[description].jpg`

Examples:
- `mobile-app-screenshot-home.png`
- `presentation-slide-features.jpg`
- `diagram-communication-protocol.png`
- `example-business-card.jpg`

## File Format Guidelines

- **Screenshots:** PNG (for clarity and transparency)
- **Diagrams:** PNG or SVG
- **Marketing:** JPEG (compressed)

## Usage

```markdown
![Mobile App Home Screen](/_assets/images/mobile-app-screenshot-home.png)
```

## Organization Tips

- Group related images with consistent prefixes
- Use descriptive filenames
- Keep file sizes optimized for web
- Update this README when adding new image categories
