# eCozy Product Photos

This directory contains product photography and lifestyle images.

## Photo Types

### Product Photography
- Isolated product shots (white background)
- Product details and close-ups
- Product in-situ (installed in real environments)

### Lifestyle Photography
- Products in modern homes
- User interaction shots
- Contextual environment photos

### Technical Photography
- Installation process
- Hardware components
- Technical details

## Naming Convention

`photo-[product]-[context]-[resolution].jpg`

Examples:
- `photo-thermostat-white-bg-2000px.jpg`
- `photo-thermostat-lifestyle-living-room-2000px.jpg`
- `photo-sensor-installation-1200px.jpg`

## Image Specifications

- **Format:** JPEG (compressed for web)
- **Resolution:** 72-96 DPI
- **Max width:** 2000px
- **File size:** < 500 KB (optimize before uploading)

## Usage

```markdown
![eCozy Thermostat](/_assets/photos/photo-thermostat-lifestyle-2000px.jpg)
*eCozy Smart Thermostat in a modern living room*
```
