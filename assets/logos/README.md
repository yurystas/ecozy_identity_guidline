# eCozy Logos

This directory contains all official eCozy logo files in various formats and variations.

## Logo Files

Place logo files here following this naming convention:

### Full Logo (Logo + Text)
- `ecozy-logo-full-color.svg` - Full color version (primary)
- `ecozy-logo-full-white.svg` - White version (for dark backgrounds)
- `ecozy-logo-full-black.svg` - Black version (for light backgrounds)
- `ecozy-logo-full-monochrome.svg` - Monochrome/grayscale version

### Logomark (Icon Only)
- `ecozy-logomark-color.svg` - Color icon only
- `ecozy-logomark-white.svg` - White icon
- `ecozy-logomark-black.svg` - Black icon

### With Tagline
- `ecozy-logo-tagline-color.svg` - Logo with tagline
- `ecozy-logo-tagline-white.svg` - White with tagline
- `ecozy-logo-tagline-black.svg` - Black with tagline

## File Formats

Each logo should be available in:
- **SVG** (preferred) - scalable, small file size
- **PNG** (2000px width) - for web use with transparency
- **PDF** - for print and professional use

## Usage

Reference in markdown:
```markdown
![eCozy Logo](/_assets/logos/ecozy-logo-full-color.svg)
```
