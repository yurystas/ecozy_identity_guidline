## About This Repository

This repository contains Markdown documentation files for Wiki.js that define the eCozy brand identity guidelines for our smart climate control ecosystem:

- **Thermostat & Temperature Sensors**: Firmware development and design standards
- **Central Unit**: Embedded Linux system architecture and implementation
- **Communication Protocols**: Inter-device standards and data formats
- **CI/CD Pipeline**: Automated testing and deployment procedures

## Usage

These Markdown files are designed to be imported into or synchronized with a Wiki.js installation for web-based browsing and editing.
